# UI/UXデザインドキュメント

## プロジェクト概要
Minimal Shopの画面デザイン全面リニューアル

**更新日**: 2025-11-05  
**バージョン**: 1.0.0

---

## 設計思想

### 1. デザインコンセプト
- **モダンでミニマル**: 無駄を省き、本質的な情報に集中
- **ユーザーフレンドリー**: 直感的な操作と分かりやすいUI
- **レスポンシブ**: あらゆるデバイスで快適な体験を提供
- **アクセシブル**: 色覚特性やスクリーンリーダーにも配慮

### 2. デザイン原則

#### a) 視覚的階層
- 大きなタイポグラフィで重要な情報を強調
- 適切な余白で読みやすさを確保
- カラーコントラストで視認性を向上

#### b) 一貫性
- 統一されたコンポーネント設計
- 予測可能なインタラクション
- 規則的なスペーシングシステム

#### c) フィードバック
- ホバー効果で操作可能要素を明示
- トランジションでスムーズな状態変化
- 空状態メッセージでユーザーをガイド

## コンポーネント設計

### 商品カード (product-card)
**構造**:
```html
<article class="product-card">
  <div class="product-image-placeholder">📦</div>
  <div class="product-info">
    <span class="product-id">ID: 1</span>
    <h3 class="product-name">商品名</h3>
    <p class="product-price">¥1,000</p>
  </div>
</article>
```

**スタイル特徴**:
- 白背景、1pxボーダー、角丸
- ホバー時: 上方向へ4px移動 + 大きなシャドウ
- トランジション: 0.2s ease
- カーソル: pointer (クリック可能を示唆)

**配置**:
- グリッドレイアウトで自動整列
- 最小幅280px、最大幅は親要素に依存
- 内部パディング: 2rem

### 空状態メッセージ (empty-state)
**目的**: 商品が存在しない場合のユーザーガイド

**構造**:
```html
<div class="empty-state">
  <div class="empty-state-icon">📭</div>
  <h3 class="empty-state-title">タイトル</h3>
  <p class="empty-state-description">説明</p>
  <a href="#" class="empty-state-link">アクション</a>
</div>
```

**スタイル特徴**:
- 点線ボーダー: 仮状態を視覚的に示す
- 中央揃え: 注目を集める
- アイコン: 直感的な理解を促進
- CTAボタン: 次のアクションを明確に提示

### ボタン (btn-admin)
**種類**:
- プライマリボタン: 青背景、白文字
- ホバー状態: 濃い青に変化

**特徴**:
- 角丸: 0.5rem
- パディング: 0.5rem 1.5rem
- トランジション: 0.2s ease
- フォントウェイト: 500 (Medium)

## レスポンシブブレークポイント

### デスクトップ (> 768px)
- フルグリッド表示
- 水平ナビゲーション
- 最大4列の商品グリッド

### タブレット (≤ 768px)
```css
@media (max-width: 768px)
```
- ヘッダーを縦方向に変更
- ナビゲーションを縦並びに
- 商品グリッド最小幅を240pxに調整

### モバイル (≤ 480px)
```css
@media (max-width: 480px)
```
- 単列グリッド
- フォントサイズ縮小
- パディング削減
- タッチフレンドリーなボタンサイズ

## インタラクション設計

### ホバーエフェクト
1. **商品カード**: 上方向に浮き上がる + シャドウ拡大
2. **ナビゲーションリンク**: 文字色変化 + 背景色追加
3. **ボタン**: 背景色を濃く変更

### トランジション
- **デュレーション**: 0.2秒（速すぎず遅すぎず）
- **イージング**: ease（自然な動き）
- **対象プロパティ**: transform, box-shadow, color, background-color

### カーソル
- **リンク・ボタン**: pointer
- **カード**: pointer（将来的な詳細ページ遷移を想定）
- **テキスト**: default

## アクセシビリティ配慮

### カラーコントラスト
- **テキスト**: WCAG AA基準を満たすコントラスト比
- **主要テキスト**: #1f2937 on #ffffff (15.8:1)
- **副次テキスト**: #6b7280 on #ffffff (4.6:1)

### セマンティックHTML
- `<header>`, `<main>`, `<footer>`: ランドマーク要素
- `<article>`: 商品カードに使用
- `<h1>`〜`<h3>`: 適切な見出し階層

### キーボードナビゲーション
- すべてのリンクとボタンがフォーカス可能
- 視覚的なフォーカスインジケーター（ブラウザデフォルト）

### 代替テキスト
- 絵文字アイコンは装飾目的（aria-label不要）
- 将来の画像追加時はalt属性を必須とする

## パフォーマンス最適化

### CSS最適化
- **カスタムプロパティ(CSS Variables)**: 一元管理、メンテナンス性向上
- **モバイルファースト**: 必要な場合のみメディアクエリ適用
- **最小限のセレクタ**: シンプルで高速なレンダリング

### フォント読み込み
- **Google Fonts**: preconnectで事前接続
- **font-display: swap**: テキストを即座に表示
- **フォールバックフォント**: システムフォントで高速表示

### 画像（将来対応）
- **遅延読み込み**: loading="lazy"属性
- **適切なフォーマット**: WebP推奨
- **レスポンシブ画像**: srcset属性
